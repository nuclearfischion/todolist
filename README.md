# the routes!

Verb         | Routes            | Description       |
------------ | ----------------- | ----------------- |
GET          | /api/todos        | List all todos    |
POST         | /api/todos        | Create a new todo |
GET          | /api/todo/:todoId | Retrieve a todo   |
PUT          | /api/todo/:todoId | Update a todo     |
DELETE       | /api/todo/:todoId | Delete a todo     |
